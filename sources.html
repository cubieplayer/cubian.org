Linux

There are two ways to get the source

  1. apt-get install linux-source
  2. git clone https://github.com/mmplayer/linux-sunxi.git -b dev/sunxi-3.4 --depth=1

Linux source files is a 7z archive which is located at /usr/src/. Use p7zip -d filename to decompress it.

Header package name is linux-headers.

Kernel Config

  • For A10 devices eg. Cubieboard1. Use config-cubian-base-sun4i
  • For A20 devices eg. Cubieboard2,Cubietruck. Use config-cubian-base-sun7i-3.4

Build Guide

  • Config
    make ARCH=arm menuconfig
  • Make
    make -j4 ARCH=arm CROSS_COMPILE=arm-none-linux-gnueabi- uImage modules
  • Install
    make STRIP=arm-none-linux-gnueabi-strip INSTALL_MOD_PATH=ROOTFS_PATH ARCH=arm INSTALL_MOD_STRIP=1 modules_install

U-Boot

Branch info

  • sunxi - A10,A20 MMC bootloader
  • sunxi-nand - A10 NAND bootloader
  • dev/a20-nand-ext4-v2 - A20 NAND bootloader

Build Guide

  • From sunxi branch
    make Cubieboard|Cubieboard2|Cubietruck CROSS_COMPILE=arm-none-linux-gnueabi-
    will build u-boot for MMC A10 and A20 boards. eg. Cubieboard 1,2 and Cubietruck
  • From sunxi-nand branch
    make Cubieboard CROSS_COMPILE=arm-none-linux-gnueabi-
    will build u-boot for NAND on A10. eg. Cubieboard1
  • From dev/a20-nand-ext4-v2 branch
    make sun7i CROSS_COMPILE=arm-none-linux-gnueabi-
    will build u-boot for NAND on A20. eg. Cubieboard2,Cubietruck
